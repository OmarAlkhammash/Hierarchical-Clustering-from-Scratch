# Hierarchical Clustering from Scratch

This project implements the **Hierarchical clustering algorithm from scratch** using only **NumPy** and **SciPy** â€” no machine learning libraries involved in the core model.

## Features
- Agglomerative clustering using:
  - Pairwise distance matrix
  - Linkage matrix
  - Flat cluster label extraction
- Core logic built using only NumPy and SciPy
- Works with multiple 2D datasets
- visualizations:
  - Scatter plots of clusters
  - Dendrograms

## Datasets

Generated using `sklearn.datasets`:
- Blobs
- Moons
- Circles
- Classification
